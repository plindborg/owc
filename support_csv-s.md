## Frequently Asked Questions

### How do I import a file?

Go to **File → Import…** and select a CSV, TSV, Excel (.xlsx), or JSON file. You can also drag and drop a file directly onto the app window.

### Which file formats are supported?

**Import:** CSV, TSV, Excel (.xlsx), JSON

**Export:** CSV, TSV, Excel (.xlsx), JSON, Markdown, HTML, SQL INSERT

### How do I export only some rows?

Select the rows you want by clicking and holding Shift or Command, then choose **File → Export As** or right-click in the table and choose Export. Only the selected rows will be included.

### How do I change a column's data type?

Right-click the column header in the sidebar and choose **Change Type**, or click the type badge next to the column name in the left sidebar.

### How do I undo a change?

Use **Edit → Undo** (⌘Z) to undo the last action. CSV Studio also keeps a version history — open it with **⌘⇧T** or the clock icon in the toolbar to restore any earlier snapshot.

### How do I take a manual snapshot?

Click the camera icon in the toolbar or go to **File → Take Snapshot** (⌥⌘S).

### How does iCloud sync work?

Save your document to iCloud Drive and it will automatically appear on your other Macs. No setup is required beyond being signed in to iCloud.

### The app isn't recognizing my CSV file correctly

Try opening the file in a text editor to check which delimiter is used (comma, semicolon, or tab). CSV Studio auto-detects the delimiter, but if the file uses an unusual format you can rename the extension to `.tsv` for tab-separated files.

### I found a bug or have a feature request

Please send an email to [support@lindborg.dev](mailto:support@lindborg.dev) with a short description. For bugs, include the file type you were working with and the macOS version you're running.
