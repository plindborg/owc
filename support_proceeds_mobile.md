## Get Started

If you are setting up Proceeds for the first time, these are the best places to begin inside the app:

- `Settings` for language, theme, currency, Pro access, and App Store Connect setup
- `Reports` for import, export, sync status, and import history
- `Overview` for your main revenue and subscription dashboard

## Quick Troubleshooting

Before contacting support, try these steps:

- Confirm that your workspace contains imported or synced data
- Open `Reports` to review recent import or sync history
- Open `Settings` to verify the current language, theme, and preferred currency
- Use `Settings > Data` to clear imported data and run a fresh import if needed
- Re-check App Store Connect credentials if sync fails
